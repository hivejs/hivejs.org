What is it?

Hive.js is a real-time collaboration platform that is modular, scalable, integratable and extensible.

It's a Node.js-based server with an accompanying web app that works in your browser.

It's a collaboration platform: it allows you to collaborate on all kinds of documents and integrate your applications with it (you don't need to use the web app).

It's modular and extensible: editors, authentication and authorization and more can be customized.

It's scalable: it allows you to spin up multiple server instances that all share the same documents.

It's integratable through a rich RESTful API and a websocket-based interface, with support for adding additional interfaces.

It's born out of the author's frustration with etherpad -- read more about its story and how it compares to etherpad.

It's still a work-in-progress and not quite perfect yet, but you can check out some demos below!
